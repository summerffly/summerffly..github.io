### C++的(*)和(&)运算符

　C++的地址运算中，(*)运算符有2种用法：

> - 声明一个指针；
> - 作为dereferencing("解引用")运算符，返回指针地址中保存的值；

　(&)运算符也有2种用法：

> - 作为address("取值")运算符，返回变量的地址；
> - 声明一个引用；

　下面用代码来展示两种运算符的用法：

```
int a = 1;   // 声明变量a，并初始化分配内存
int *p = &a;   // 声明指针p，用a的地址初始化
int &c = *p;   // 声明引用c，不能再次赋值
```
